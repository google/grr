How to create a new fleet collection?

Fleet collections can be created from an existing flow or fleet collection.

From a flow

  1. From the client page, click the more_vert button on the card and select "Create a fleet collection".

From another hunt

  1. From the fleet collection page, click "Copy and tweak collection".
  2. From the fleet collection lists page, click the more_vert button on the card and select "Duplicate collection".
